# Website Development Prompt

Build a high-end fashion e‑commerce website similar to **Calvin Klein**
using **Sanity Studio** as the headless CMS and **Next.js** as the
frontend framework. The website should include:

## Key Requirements

-   Modern, minimal, high-fashion design style.
-   Fully responsive layouts.
-   Product catalog with categories, variants, pricing, and image
    galleries.
-   Home page with hero banners, featured collections, promos.
-   Fast-loading images using Sanity's Image CDN.
-   Shopping cart with add/remove/update functions.
-   Checkout flow integrated with Stripe, Paystack, or Mobile Money.
-   Reusable UI components (buttons, cards, product tiles, navbars).
-   SEO‑optimized pages.
-   Headless CMS structure in Sanity:
    -   `product`
    -   `category`
    -   `homePage`
    -   `siteSettings`
    -   `banner`

## Technical Stack

-   **Sanity Studio v3** (for content management)
-   **GROQ** queries (for lightning-fast data fetching)
-   **Next.js App Router**
-   **TailwindCSS + shadcn/ui**
-   **Stripe/Paystack API**
-   **Vercel deployment**

## Deliverables

-   Complete Sanity schema definitions.
-   Next.js frontend project structure.
-   Dynamic product listing and product detail pages.
-   Checkout API routes.
-   Global navigation and footer.
-   CMS-driven homepage.
-   Deployment instructions.

Use clean, scalable architecture and production‑ready code.
